# 👟 SyntaxWear - E-commerce de Sneakers

![SyntaxWear Banner](./images/banners/hero.jpg)

A **SyntaxWear** é uma landing page de um e-commerce moderno e responsivo focado na venda de tênis e sneakers exclusivos. O projeto foi desenvolvido com foco em performance, acessibilidade e design contemporâneo.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end:

- **HTML5**: Estrutura semântica para melhor SEO e acessibilidade.
- **CSS3**: Utilização de variáveis, Flexbox e Grid Layout para um design flexível.
- **Metodologia de Componentes**: Organização do CSS por componentes para facilitar a manutenção.
- **Google Fonts**: Integração com a fonte 'Ubuntu' para uma tipografia moderna.

## 📁 Estrutura do Projeto

A organização dos arquivos segue um padrão modular:

```text
/
├── index.html          # Página principal do site
├── css/                # Estilização
│   ├── base.css        # Estilos globais e reset
│   ├── variables.css   # Definição de cores e fontes
│   └── components/     # CSS modularizado por seção (header, footer, hero, etc.)
├── images/             # Ativos visuais
│   ├── banners/        # Imagens principais de destaque
│   ├── icons/          # Ícones em SVG
│   ├── logo/           # Logotipo da marca
│   └── products/       # Fotos dos produtos
└── js/                 # Scripts (reservado para futuras implementações)
```

## ✨ Funcionalidades e Seções

- **Header Responsivo**: Menu de navegação com suporte a dispositivos móveis (hamburguer menu).
- **Seção Hero**: Destaque principal para o modelo "Krypton One" com chamadas para ação (CTA).
- **Categorias**: Navegação rápida por estilos (Casual, Esporte, Moderno e Futurista).
- **Grid de Produtos**: Exibição dinâmica de produtos com cards estilizados.
- **Newsletter**: Área para captura de e-mails integrada ao rodapé.
- **Rodapé Completo**: Links de navegação, redes sociais e informações de copyright.
